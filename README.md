# Project 3: Traveling Salesman Problem

A C++ project for Data Structures and Algorithms II that solves the Traveling Salesman Problem using both brute force and a genetic algorithm.

The program reads city distance data from a text file, prompts the user for algorithm settings, runs the selected computations, and prints the best tour results and runtimes to the console.

Please refer to **UsersManual.pdf** in the root of this project for more detailed information and how to run the program.
